FBOCC
-----

fbocc.cpp contains the program corresponding to FBOCC locking algorithm. 

To compile : 
g++ fbocc.cpp -std=c++11 -lpthread

To run: 
./a.out <inputfile> > <outputfile>

Eg. ./a.out input.txt > output.txt

This redirects the output to the file output.txt

Example input has been attached in the folder.
